# Elm development
This is a simple `make` and `Docker`-based development environment for playing with [#Elm-lang](http://elm-lang.org/). The image is based on [node:5-slim](https://hub.docker.com/_/node/)

To bring up a REPL, simply:
```bash
	make build repl
```
